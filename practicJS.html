Задание0

true + false
Ответ: 1
Потому что оператор + преобразует операнды в числа перед выполнением сложения, true становится 1, false становится 0, поэтому 1 + 0 = 1

12 / "6"
Ответ: 2
Потому что оператор / также преобразует операнды в числа, "6" становится 6, поэтому 12 / 6 = 2

"number" + 15 + 3
Ответ: "number153"
Поскольку оператор + используется для объединения строк, все операнды преобразуются в строки и склеиваются вместе

15 + 3 + "number"
Ответ: "18number"
Здесь сначала выполняется сложение чисел, затем результат преобразуется в строку и объединяется со строкой "number"

'1' > null
Ответ: true
Здесь строка '1' преобразуется в число, а null преобразуется в 0, поэтому выражение становится 1 > 0 и возвращает true

"foo" + + "bar"
Ответ: "fooNaN"
Здесь оператор + используется для конкатенации строк, но после первой строки есть + перед строкой "bar", который преобразует ее в NaN (нечисловое значение), поэтому результат будет "fooNaN"

'true' == true
Ответ: false
Поскольку это сравнение с участием ==, строки 'true' не равны булевому значению true, поэтому результат будет false

false == 'false'
Ответ: false
Аналогично предыдущему случаю, здесь false не равно строке 'false', поэтому результат будет false

null == ''
Ответ: false
Потому что null может быть равен только null или undefined, но не пустой строке
